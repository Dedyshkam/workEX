# 🧠 Еволюція Штучного Інтелекту

Сучасний лендинг про розвиток ІІ від зародження до наших днів з повноцінним backend на Flask.

## 🌟 Можливості

- 📜 Історія ІІ з візуальною хронологією (від Тьюринга до ChatGPT)
- 🎨 Приклади застосування ІІ у фото, мистецтві, відео
- 🖼️ Галерея нейромережевих шедеврів з еволюцією якості
- 📊 Інтерактивне опитування із збереженням відповідей до бази даних
- ✨ Анімації, плавний скролл, адаптивний дизайн
- 🐍 Backend на Flask + SQLite
